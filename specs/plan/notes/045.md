## NC2: Test mixed-error fallback in _handle_cancel_remote_job

The mixed-error fallback added by WI-040 in `_handle_cancel_remote_job`. The third pass (after conflict pass and auth pass) handles the case where some workers had connection errors and others returned 404:

```python
if error_workers and not_found_workers:
    # combined message with both details
```

Test strategy: configure two workers, mock `_cancel_one` so:
- Worker A returns `{"_status": "error", "_msg": "Connection refused"}`
- Worker B returns `{"_status": "not_found"}`

Call `_handle_cancel_remote_job({"job_id": "some-id"})`, verify the response contains error text referencing both the not-found and connection error details.

```python
async def test_cancel_remote_job_mixed_error_and_not_found(tmp_working_dir):
    spawner._remote_workers = [
        {"name": "w1", "url": "http://w1", "api_key": "k1"},
        {"name": "w2", "url": "http://w2", "api_key": "k2"},
    ]

    async def mock_cancel_one(worker, job_id, session):
        if worker["name"] == "w1":
            return {"_status": "error", "_msg": "Connection refused"}
        return {"_status": "not_found"}

    with patch.object(spawner, "_cancel_one", side_effect=mock_cancel_one):
        result = await spawner._handle_cancel_remote_job({"job_id": "test-job-id"})

    data = json.loads(result[0].text)
    assert "error" in data
    # Should mention both not-found workers and connection error details
    assert "w2" in data["error"] or "not found" in data["error"].lower()
    assert "Connection refused" in data["error"] or "w1" in data["error"]
```

## CF7: Test MCP tool schema validation

`list_tools()` is an async function at the top of session-spawner/server.py (line 55). It returns a list of `Tool` objects, each with `name`, `description`, and `inputSchema`.

Test strategy: call `await spawner.list_tools()` and verify each tool has a `required` array in its `inputSchema` containing the expected fields.

```python
async def test_tool_schemas_have_required_fields():
    tools = await spawner.list_tools()
    by_name = {t.name: t for t in tools}

    schema = by_name["spawn_session"].inputSchema
    assert "required" in schema
    assert "prompt" in schema["required"]
    assert "working_dir" in schema["required"]

    schema = by_name["spawn_remote_session"].inputSchema
    assert "required" in schema
    assert "prompt" in schema["required"]
    assert "working_dir" in schema["required"]

    schema = by_name["poll_remote_job"].inputSchema
    assert "required" in schema
    assert "job_id" in schema["required"]

    schema = by_name["cancel_remote_job"].inputSchema
    assert "required" in schema
    assert "job_id" in schema["required"]

    schema = by_name["list_remote_workers"].inputSchema
    assert "required" in schema or schema.get("required", []) == []  # list_remote_workers has no required fields
```
